## UStruct
- 구조체 이름은 F 접두사
- NewObject API 사용 불가. 대부분의 경우 스택 내 데이터로 사용
- 선언 위에 `USTRUCT()`, 본문 `GNERATED_BODY()`, 블루프린트에 사용하기 `UPROPERTY()`

## TArry

- 요소 순서대로 저장, 중복 허용, 빠른 인덱스 접근
- 접근O(1), 검색 O(N) , 삽입/삭제 O(N)

## TSet

- 순서 없음, 중복 비허용, 빠른 검색,
- 접근O(1), 검색 O(1) , 삽입/삭제 O(1)
- 내부적으로 TSparseArray를 기반한 해시테이블
